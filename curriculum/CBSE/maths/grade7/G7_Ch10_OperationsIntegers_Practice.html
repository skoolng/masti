Set A Basic – Placing & Ordering Integers

Plot −5, −2, 0, 3, 7 on a number line. Which integer is furthest from zero?
Easy
Answer

Answer: 7

Method: Distance from Zero (Absolute Value)
Step 1: Distance from zero = absolute value: |−5|=5, |−2|=2, |0|=0, |3|=3, |7|=7.
Step 2: Compare: 7 > 5 > 3 > 2 > 0.
Step 3: The largest absolute value is 7, so 7 is furthest from zero.
Key Rule: The distance of any integer from zero equals its absolute value, regardless of sign.
Write all integers from −4 to 4 in ascending order.
Easy
Answer

Answer: −4, −3, −2, −1, 0, 1, 2, 3, 4

Method: Ascending = Left to Right on Number Line
Step 1: On a number line, numbers increase from left to right.
Step 2: Start from the most negative: −4.
Step 3: Move right: −3, −2, −1, 0, 1, 2, 3, 4.
Key Rule: Ascending order means smallest first; on a number line, left = smaller.
Which is greater: −8 or −3?
Easy
Answer

Answer: −3

Method: Number Line Comparison
Step 1: On a number line, −3 is to the right of −8.
Step 2: Numbers to the right are always greater.
Step 3: Therefore −3 > −8.
Key Rule: Among negative numbers, the one closer to zero is greater (less negative = greater).

If a < 0 and b > 0 and |a| > |b|, is a + b positive or negative? Give an example.
Hard
Answer

Answer: Negative (e.g., −7 + 3 = −4)

Method: Sign Dominance Rule
Step 1: a is negative, b is positive. Their sum's sign depends on which has the larger absolute value.
Step 2: Given |a| > |b|, the negative number "wins" — the sum is negative.
Step 3: Example: a = −7, b = 3, |−7|=7 > |3|=3. Sum = −7+3 = −4 (negative).
Key Rule: When adding integers of opposite signs, the sign of the result matches the integer with the larger absolute value.
