Name the four main tools used in geometric construction.
Easy
Answer

Compass, ruler (straight edge), protractor, set square

Step 1: Compass: draws circles and arcs; measures equal distances.
Step 2: Ruler: draws straight lines; measures lengths.
Step 3: Protractor: measures and draws angles.
Step 4: Set square: draws perpendicular and parallel lines.
Key Rule: Construction uses compass, ruler, protractor, and set square.

Compass and straightedge use only circles and straight lines — proven exact by Euclidean axioms
