# English MYP4-5 - Chapter-wise Practice Sets

Source: Book(s) in /english/myp4-5 folder

Format used across all chapters: **4 sets per chapter** (Set A, Set B, Set C, Set D).

## Chapter 1: Where do I belong?

### Video Resources
- [Khan Academy search](https://www.google.com/search?q=Khan+Academy+english+Where+do+I+belong%3F)
- [YouTube lesson search](https://www.youtube.com/results?search_query=english+Where+do+I+belong%3F+lesson)
- [IB MYP topic video search](https://www.google.com/search?q=IB+MYP+english+Where+do+I+belong%3F+revision)

### Set A - Foundation
1. Read a short text linked to **Where do I belong?** and identify the main idea and 3 supporting details.
2. Find 8 key vocabulary items from the text and write learner-friendly meanings.
3. Answer 5 comprehension questions (literal + inferential) from the same text.
4. Identify tone, audience, and purpose; justify each with textual evidence.
5. Summarize the text in 80-100 words using your own words.

### Set B - Core Skills
1. Create 10 grammar-edit sentences inspired by **Where do I belong?** and correct them.
2. Rewrite 6 sentences using richer connectors and varied sentence structures.
3. Transform direct speech to reported speech (or vice versa) for 8 lines of dialogue.
4. Do a word-choice upgrade: replace 12 basic words with precise alternatives.
5. Write a short paragraph using at least 6 target vocabulary words accurately.

### Set C - Application
1. Write one guided paragraph and one extended response connected to **Where do I belong?**.
2. Plan and draft a 250-300 word response with clear structure: intro, body, conclusion.
3. Use one persuasive, one descriptive, and one reflective writing technique in your draft.
4. Add peer-feedback comments on clarity, coherence, and language accuracy.
5. Revise your writing and submit a second draft with 5 specific improvements.

### Set D - Challenge
1. Prepare a 2-minute speaking response on **Where do I belong?** with opening, evidence, and closing.
2. Listen to one related video and note 6 key points + 3 questions you still have.
3. Run a mini debate: write arguments for and against a chapter-related statement.
4. Create an oral presentation outline with visuals, transitions, and timing.
5. Complete a reflection: strengths, language targets, and next-step action plan.
