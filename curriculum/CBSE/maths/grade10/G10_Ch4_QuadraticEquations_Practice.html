For what values of k is (k+1)x² + kx + 1 = 0 a quadratic equation?Medium

Expand and write in standard form: (x – 1/x)² = 3.Hard

If x = 1 and x = –1 are both roots of ax² + bx + c = 0, what can you say about a, b, c?Medium

Is x = 0 ever a root of ax² + bx + c = 0 when c ≠ 0?Hard
Show Answer

Substitute x=0: a(0)+b(0)+c=c. For x=0 to be a root, c=0. So x=0 is never a root when c≠0.

If x = p and x = q are roots of ax² + bx + c = 0, verify that a(x–p)(x–q) = ax² + bx + c.Hard
Show Answer

a(x–p)(x–q)=a[x²–(p+q)x+pq]=ax²–a(p+q)x+apq. Compare: b=–a(p+q) and c=apq. These are Vieta's formulas, and they are indeed correct for the factored form.

Find a, b if x = 3 and x = –5 are roots of x² + bx + a = 0.Hard
Show Answer

Show that if p is a root of ax² + bx + c = 0, then so is –b/a – p (the other root).Hard
Show Answer

Sum of roots = –b/a. If p is one root, other root = –b/a – p. This follows from Vieta's formula: α+β=–b/a, so β=–b/a–α=–b/a–p.

SET D Properties and Classification

How many roots can a quadratic equation have at most?Easy
Show Answer

At most 2 roots (degree 2). It can have 2 distinct real roots, 1 repeated real root, or 2 complex (no real) roots.

Why must a ≠ 0 in ax² + bx + c = 0 for it to be quadratic?Easy
Show Answer

If a=0, the x² term vanishes and the equation becomes bx+c=0, which is linear (degree 1), not quadratic (degree 2).

Give one example each of a quadratic equation with: (i) two rational roots (ii) two irrational roots (iii) no real roots.Easy
Show Answer

(i) x²–5x+6=0, roots 2,3. (ii) x²–2=0, roots ±√2. (iii) x²+1=0, roots ±i (not real).

If ax² + bx + c = 0 and –ax² – bx – c = 0 are two equations, do they have the same roots?Medium
Show Answer

Yes. Multiplying an equation by –1 gives an equivalent equation with the same roots: –(ax²+bx+c)=0 iff ax²+bx+c=0.

Can a quadratic equation have exactly one real root and one non-real root?Medium
Show Answer

No. Complex roots of polynomials with real coefficients always come in conjugate pairs. So either both roots are real, or both are non-real complex.

State the relationship between roots α, β and coefficients a, b, c of ax²+bx+c=0 (Vieta's formulas).Medium
Show Answer

Sum: α+β = –b/a. Product: αβ = c/a.

If the roots of x²+px+q=0 are doubled, write the new equation.Hard
Show Answer

New roots: 2α, 2β. Sum=2(α+β)=–2p. Product=4αβ=4q. New equation: x²+2px+4q=0.

If α, β are roots of 2x²–3x+1=0, find α²+β².Hard
Show Answer

α+β=3/2, αβ=1/2. α²+β²=(α+β)²–2αβ=9/4–1=5/4.

The product of the roots of 5x² – kx + 20 = 0 is 4. Find k and the sum of roots.Hard
Show Answer

Product=20/5=4 ✓ (holds for any k). Sum=k/5. We cannot determine k from product alone. Need more information.

If α and β are roots of x²–x–1=0, find α³+β³.Hard
Show Answer

α+β=1, αβ=–1. α³+β³=(α+β)³–3αβ(α+β)=1–3(–1)(1)=1+3=4.
