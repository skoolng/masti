Set A What Is a Function?

Define a function. How is it different from a general relation?
Show Answer

A function is a relation where every input (x-value) maps to exactly one output (y-value). A relation may map one input to multiple outputs, which would disqualify it as a function.

Easy
Is the mapping {(1,3), (2,5), (3,3), (4,7)} a function? Justify.
Show Answer

Yes. Each x-value (1, 2, 3, 4) appears exactly once, so every input has a unique output. Different inputs can share the same output — that is allowed.

Easy
Is the mapping {(1,2), (1,4), (2,3)} a function? Why or why not?
Show Answer

No. The input x = 1 maps to two different outputs (2 and 4), violating the one-output rule.

Easy
Explain the vertical line test. Draw a sketch to support your answer.
Show Answer

If any vertical line crosses a graph at more than one point, the graph is not a function. A circle fails the test (a vertical line through its interior hits two points); a straight line passes it.

Easy
Which of the following is NOT a function? (A) y = 3x + 1 (B) y = x² (C) x = y² (D) y = |x|
Show Answer

C: x = y². For x = 4, y = ±2, so one input gives two outputs.

Easy
A machine takes any number and doubles it. Write this as a function using f(x) notation.
Show Answer

f(x) = 2x

Easy
If f(x) = 5x − 3, find f(2) and f(0).
Show Answer

f(2) = 5(2) − 3 = 10 − 3 = 7; f(0) = 5(0) − 3 = −3

Easy
If g(x) = x² + 1, find g(3) and g(−3). What do you notice?
Show Answer

g(3) = 9 + 1 = 10; g(−3) = 9 + 1 = 10. Equal outputs for opposite inputs — g is an even function.

Medium
A relation is given by the table: x: 0, 1, 2, 3 → y: 4, 7, 4, 9. Is it a function?
Show Answer

Yes. Each x-value appears only once; it doesn't matter that y = 4 appears twice.

Medium
True or false: every function is a relation, but not every relation is a function. Explain.
Show Answer

True. A function is a special relation with the single-output rule. Relations that violate this (e.g., a circle) are not functions.

Medium

Set C Identifying Functions from Graphs and Tables

A graph is a straight line through the origin. Is it a function?
Show Answer

Yes. Any non-vertical straight line passes the vertical line test.

Easy
A circle centred at the origin with radius 5 — is it a function? Use the vertical line test.
Show Answer

No. A vertical line through x = 0 (for instance) intersects the circle at (0, 5) and (0, −5) — two points.

Easy
Table: x → −2, 0, 2, 4 and y → 5, 5, 5, 5. Is this a function?
Show Answer

Yes — it is a constant function f(x) = 5. Each input maps to exactly one output.

Easy
A graph looks like a parabola opening upward. Does it represent a function?
Show Answer

Yes. A vertical line can hit a parabola (opening up or down) at most once, so it passes the test.

Easy
Is y = √x a function? What restriction applies?
Show Answer

Yes, provided we take only the principal (non-negative) square root. The domain is x ≥ 0 and range is y ≥ 0.

Medium
The mapping diagram shows: 1→a, 2→b, 2→c, 3→d. Is it a function?
Show Answer

No. Input 2 maps to both b and c, violating the single-output rule.

Medium
Can two different inputs share the same output and still form a function?
Show Answer

Yes. For example f(2) = 4 and f(−2) = 4 is perfectly valid — the rule only says each input has one output, not that outputs must be unique.

Medium
A graph has a horizontal line y = 3. Does it represent a function?
Show Answer

Yes. Every x maps to exactly one y-value (3). It is a constant function.

Medium
Explain why a vertical line x = 2 does NOT represent a function.
Show Answer

For x = 2, there are infinitely many y-values (the entire vertical line). One input maps to infinitely many outputs — not a function.

Hard
Does the graph of y = |x| represent a function? Describe its shape.
Show Answer

Yes. y = |x| is a V-shaped graph with vertex at the origin; the right arm has slope +1 and the left arm slope −1. Every x gives exactly one y.

Set A Understanding Domain and Range

Define domain and range of a function.
Show Answer

Domain: the set of all valid input (x) values. Range: the set of all resulting output (y) values.

Easy
For f(x) = {(1,3),(2,5),(3,7)}, state the domain and range.
Show Answer

Domain = {1, 2, 3}; Range = {3, 5, 7}

Easy
What is the domain of f(x) = x + 4? (no restrictions)
Show Answer

All real numbers, written ℝ or (−∞, +∞). Any x-value is valid.

Easy
What is the domain of f(x) = 1/(x−3)?
Show Answer

x − 3 ≠ 0, so x ≠ 3. Domain: all real numbers except 3.

Easy
What is the domain of f(x) = √(x − 5)?
Show Answer

x − 5 ≥ 0 → x ≥ 5. Domain: [5, ∞).

Medium
For y = x², state the domain and range.
Show Answer

Domain: all real numbers (−∞, ∞). Range: y ≥ 0, i.e. [0, ∞) since squares are never negative.

Easy
For f(x) = −x², state the range.
Show Answer

y ≤ 0. Range: (−∞, 0].

Medium
A function has domain {−2, 0, 2, 4} and rule f(x) = 2x + 1. Find the range.
Show Answer

f(−2)=−3, f(0)=1, f(2)=5, f(4)=9. Range = {−3, 1, 5, 9}

Medium
For the graph y = √x (x ≥ 0), what is the range?
Show Answer

y ≥ 0, i.e. [0, ∞).

Medium
Can the range of a function be a single number? Give an example.
Show Answer

Yes — a constant function like f(x) = 7 has range {7}.

Easy

Set A Slope and Intercepts

Write the slope-intercept form of a linear function and label each part.
Show Answer

y = mx + c, where m is the slope (gradient) and c is the y-intercept.

Easy
Find the slope and y-intercept of y = 4x − 7.
Show Answer

Slope m = 4; y-intercept c = −7 (point (0, −7)).

Easy
What is the slope of a horizontal line y = 5?
Show Answer

Slope = 0. Horizontal lines are flat — no rise for any run.

Easy
Find the slope between points (2, 3) and (6, 11).
Show Answer

m = (11−3)/(6−2) = 8/4 = 2

Easy
A line has slope −3 and passes through (0, 4). Write the equation.
Show Answer

y = −3x + 4

Easy
Find the x-intercept of y = 2x − 6.
Show Answer

Set y = 0: 0 = 2x − 6 → x = 3. x-intercept is (3, 0).

Medium
Write in slope-intercept form: 3x + 2y = 12.
Show Answer

2y = −3x + 12 → y = −(3/2)x + 6. Slope: −3/2; y-intercept: 6.

Medium
Two points on a line are (1, 5) and (3, 11). Find the equation of the line.
Show Answer

m = (11−5)/(3−1) = 3. y − 5 = 3(x−1) → y = 3x + 2.

Medium
Are the lines y = 2x + 1 and y = 2x − 5 parallel? Explain.
Show Answer

Yes. Both have slope 2 but different y-intercepts — parallel lines never meet.

Medium
Line A has slope 3 and line B has slope −1/3. Are they perpendicular?
Show Answer

Yes. Product of slopes = 3 × (−1/3) = −1, confirming perpendicularity.
