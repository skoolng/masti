SET A Conceptual – Identifying Variables

A letter that stands for an unknown or changing number is called a ___________.Easy
Show Answer

Variable (letter-number).

A variable is a symbol (usually a letter like x, n, a) used to represent a value that can change or is unknown. Key Rule: Variables can take different values in different situations.

Which of the following is a variable: 7, x, 15, 100?Easy
Show Answer

x

7, 15, and 100 are fixed numbers (constants). x is a letter representing an unknown value. Key Rule: Variables are letters; constants are fixed numbers.

Riya has some marbles. She doesn't know how many. Which letter can represent her marbles?Easy
Show Answer

Any letter, e.g., m (for marbles)

We can write the number of marbles as m. m stands for an unknown quantity. Key Rule: Choose a meaningful letter — m for marbles, n for number, p for price etc.

The temperature today is unknown. Write a variable expression for the temperature.Easy
Show Answer

t (or any letter)

Let t = today's temperature. Since the value is unknown, we use a letter. Key Rule: A variable holds the place of a number we don't yet know.

In the expression 5n, which part is the variable?Easy
Show Answer

n

5 is the fixed multiplier (coefficient). n is the letter-number (variable) that can change. Key Rule: The letter in a product is always the variable.

A bag has n apples. If n = 6, how many apples are in the bag?Easy
Show Answer

6 apples

Replace n with 6. The bag has 6 apples. Key Rule: Variables become numbers when we substitute a specific value.

Can the same variable take different values in different problems?Medium
Show Answer

Yes

In one problem x = 3; in another x = 10. The letter x doesn't have a single permanent value. Key Rule: A variable's value is set by the context or given information of each problem.

Write a variable to represent: "the number of students absent today."Easy
Show Answer

a (or any letter)

Let a = number of students absent. Key Rule: Choosing a letter related to the context (a for absent) makes expressions easier to read.

True or False: The variable x always equals 5.Medium
Show Answer

False

x can be any value depending on the equation or situation. Key Rule: Variables are not fixed; they can represent any number.

A train travels at a speed that changes each day. What kind of quantity is the speed?Medium
Show Answer

A variable quantity

Because the speed changes, it is best represented by a letter-number (variable) like s. Key Rule: Any changing or unknown quantity can be named with a variable.

Remaining = original − given = p − 5. Key Rule: "Gave away" or "lost" means subtraction.

A box has w sweets. If there are 4 boxes, write an expression for the total sweets.Easy

4w

Total = 4 × w = 4w. Key Rule: Repeated addition of w four times = 4w.

A ribbon of length L cm is cut into 3 equal pieces. Write an expression for each piece.Medium
Show Answer

L ÷ 3 or L/3

Length of each piece = total ÷ 3 = L/3. Key Rule: Division is written as a fraction in algebra.

A shop sells k kg of rice each day. Write an expression for rice sold in 7 days.Medium

7k

7 days × k kg per day = 7k kg. Key Rule: Repeated multiplication gives a coefficient in front of the variable.

Meena scored s marks in Maths and 12 marks in Science. Write the total.Easy

Total = Maths + Science = s + 12. Key Rule: Combine the known number and the variable with the correct operation.

A rectangle has width w. Its length is twice its width. Write an expression for the length.Medium

2w

Length = 2 × width = 2w. Key Rule: "Twice" means multiply by 2.

A tank holds T litres. After using 20 litres, write the expression for remaining water.Medium

Remaining = T − 20 litres. Key Rule: "After using" or "after removing" signals subtraction from the variable.

SET C Mixed Practice – Variables in Context

Identify the variable in: 9a − 4Easy
Show Answer

a

9 is the coefficient, 4 is the constant, and a is the variable (letter-number). Key Rule: The letter in the expression is always the variable.

Write three different variables you might use in a maths problem.Easy
Show Answer

e.g., x, n, t (many correct answers)

Common choices: x (unknown), n (number), t (time), a (age), d (distance). Key Rule: Any letter can be a variable.

If age = a years now, what is the age 5 years later?Easy
Show Answer

a + 5

Future age = current age + 5 = a + 5. Key Rule: Adding to a variable shifts it by a fixed amount.

A number n is multiplied by itself. Write the expression.Medium
Show Answer

n × n = n²

n multiplied by n is written n² (n squared). Key Rule: n² means n × n.

A car travels at v km/h for 3 hours. Write the distance travelled.Medium
Show Answer

3v km

Distance = speed × time = v × 3 = 3v. Key Rule: Distance = Speed × Time → d = 3v.

Can two different letters in the same expression represent the same value? For example, x + y where x = y = 4?Medium
Show Answer

Yes, it is possible

x and y can both equal 4 in a specific problem, making x + y = 8. Key Rule: Different letters may happen to have equal values, but they still represent different unknowns unless stated otherwise.

There are n rows of seats with 5 seats each, plus 3 extra seats. Write the total seats.Medium
Show Answer

5n + 3

Rows contribute 5n seats; add 3 extra = 5n + 3. Key Rule: Combine the variable part and constant part with +.

Explain why we use a letter for an unknown number instead of leaving a blank space.Hard
Show Answer

Letters allow us to write and manipulate rules and equations clearly.

A blank space cannot be added or multiplied; a letter can. Writing 3x + 5 = 17 lets us solve for x using algebraic rules. Key Rule: Variables make generalisation and calculation possible.

A number is halved and then 1 is added. If the number is n, write the expression.Medium
Show Answer

n/2 + 1

Halved → n/2; then add 1 → n/2 + 1. Key Rule: Follow the order of operations described in words.

A number x is decreased by 7. The result is multiplied by 2. Write the expression.Hard
Show Answer

2(x − 7)

Step 1: decrease x by 7 → (x − 7). Step 2: multiply by 2 → 2(x − 7). Key Rule: Use brackets to show that the subtraction happens before the multiplication.

SET D Challenge – Abstract Reasoning

If n = 0, what does the expression 6n equal?Easy
Show Answer

0

6n = 6 × 0 = 0. Key Rule: Any number multiplied by 0 equals 0.

If n = 1, what does 5n + 2 equal?Easy
Show Answer

7

5(1) + 2 = 5 + 2 = 7. Key Rule: Substitute the value, then evaluate.

Write an expression where the variable is t and the answer is always 10 more than t.Medium
Show Answer

t + 10

Adding 10 to any value of t gives a result 10 more. Key Rule: "More than" translates to addition.

Two brothers have ages a and b. Their combined age is 30. Write this as an equation.Medium
Show Answer

a + b = 30

Sum of ages = 30, so a + b = 30. Key Rule: An equation states that two expressions are equal.

Deepa thinks of a number x. She doubles it and subtracts 3. Write the expression.Medium
Show Answer

2x − 3

Double x → 2x; subtract 3 → 2x − 3. Key Rule: Translate each operation in order into algebra.

Three consecutive numbers starting from n. Write expressions for all three.Medium
Show Answer

n, n+1, n+2

Consecutive means each number is 1 more than the previous. Key Rule: Consecutive integers differ by 1.

The sum of two consecutive numbers starting from n is 25. Write the equation.Hard
Show Answer

n + (n + 1) = 25, i.e., 2n + 1 = 25

Sum = n + n + 1 = 2n + 1. Set equal to 25: 2n + 1 = 25. Key Rule: Combine like terms (n + n = 2n) first.

If x = 3 makes the expression 4x − k equal to 10, find k.Hard
Show Answer

k = 2

4(3) − k = 10 → 12 − k = 10 → k = 2. Key Rule: Substitute the known value, then solve for the unknown.

If the perimeter of a regular hexagon is P and each side is s, write the relationship between P and s.Hard
Show Answer

P = 6s

A regular hexagon has 6 equal sides. P = 6 × s = 6s. Key Rule: Perimeter of a regular polygon = number of sides × side length.

A number q is added to its double. Is the result always even, always odd, or neither?Hard
Show Answer

Always equal to 3q (not necessarily always even or always odd)

q + 2q = 3q. If q = 2, result = 6 (even). If q = 3, result = 9 (odd). So it depends on q. Key Rule: Simplify algebraically first, then test values.

Write an expression: "5 more than a number n."Easy

Write an expression: "a number x decreased by 8."Easy

x − 8

Decreased by → subtract → x − 8. Key Rule: "Decreased by / less than" = subtraction.

Write an expression: "4 times a number a."Easy
Show Answer

4a

4 times a = 4 × a = 4a. Key Rule: "Times" = multiplication; write the coefficient before the variable.

Write an expression: "a number b divided by 6."Easy
Show Answer

b/6

b divided by 6 = b ÷ 6 = b/6. Key Rule: Division is written as a fraction in algebra.

Write an expression: "3 less than twice a number n."Medium
Show Answer

2n − 3

Twice n = 2n; 3 less than that = 2n − 3. Key Rule: Do the multiplication first, then subtract the constant.

Write an expression: "the sum of a number x and its triple."Medium
Show Answer

x + 3x = 4x

Triple of x = 3x; sum = x + 3x = 4x. Key Rule: Combine like terms: x + 3x = 4x.

Write an expression: "7 more than 3 times a number t."Medium
Show Answer

3t + 7

3 times t = 3t; 7 more = 3t + 7. Key Rule: The variable part comes first by convention.

Write an expression: "a number n multiplied by itself, minus 5."Medium
Show Answer

n² − 5

n × n = n²; minus 5 = n² − 5. Key Rule: n² means n squared.

Write an expression: "one-third of a number y."Medium
Show Answer

y/3 or (1/3)y

One-third means divide by 3 = y/3. Key Rule: "One-third of" = × (1/3) = ÷ 3.

Write an expression: "the product of two numbers p and q, increased by 4."Hard
Show Answer

pq + 4

Product of p and q = pq (or p × q); increased by 4 = pq + 4. Key Rule: When two variables are multiplied, write them side by side: pq.

SET A Identifying Parts

In 7x + 3, identify the coefficient of x.Easy
Show Answer

7

7x means 7 multiplied by x. The multiplier 7 is the coefficient. Key Rule: Coefficient is the number multiplying the variable.

In 7x + 3, identify the constant.Easy
Show Answer

3

3 stands alone with no variable attached to it. It is a constant. Key Rule: A constant is a number that never changes in the expression.

How many terms does 5a + 2b − 9 have?Easy
Show Answer

3 terms: 5a, 2b, and −9

Terms are separated by + or −. Count: 5a | 2b | −9 = 3 terms. Key Rule: Terms are separated by addition or subtraction signs.

In 4n − 6, what is the coefficient of n?Easy
Show Answer

4

4n = 4 × n, so the coefficient is 4. Key Rule: Even with subtraction, the coefficient of the variable term is still positive unless the variable term is negative (e.g., −4n has coefficient −4).

In x + 5, what is the coefficient of x?Medium
Show Answer

1

x alone means 1 × x = x. The coefficient is 1 (invisible but present). Key Rule: When no number is written before a variable, the coefficient is 1.

In −3y + 8, what is the coefficient of y?Medium
Show Answer

−3

−3y means −3 × y. The coefficient is −3 (negative). Key Rule: A negative sign is part of the coefficient.

Name all three types of components that can appear in an algebraic expression.Easy
Show Answer

Variables, coefficients, and constants

Variable: the letter (x, n…); Coefficient: the number multiplying the variable; Constant: a standalone number. Key Rule: These three building blocks make up all algebraic expressions.

In 9 − 2x, identify the constant and the coefficient.Medium
Show Answer

Constant = 9, coefficient of x = −2

9 stands alone (constant); −2x has coefficient −2. Key Rule: The sign immediately before the term belongs to that term.

How many terms are in 3a?Easy
Show Answer

1 term (a monomial)

There is only one term: 3a. No addition or subtraction separates it. Key Rule: One term = monomial; two terms = binomial; three terms = trinomial.

In 6p + 4q − 7, identify: (a) number of terms (b) coefficient of q (c) constant.Medium
Show Answer

(a) 3 terms (b) 4 (c) −7

(a) 6p, 4q, −7 → 3 terms. (b) 4q → coefficient = 4. (c) −7 is standalone = constant. Key Rule: Constant is the term with no variable.

What is the coefficient of the constant in any expression?Hard
Show Answer

A constant is its own value — it doesn't have a "coefficient" in the traditional sense.

In 3x + 5, the 5 is a constant. We can think of it as 5 × x⁰ = 5 × 1 = 5. Key Rule: Constants are standalone numbers; coefficients attach to variables.

In 10p − 3q + 0, what is the constant?Medium

0

Write the expression that has coefficient 6 for x, coefficient 2 for y, and no constant.Easy

No constant means no standalone number. Key Rule: An expression without a constant has only variable terms.

In the expression 4m² + 3m − 7, name every term and its type.Medium

4m² (variable term, coefficient 4), 3m (variable term, coefficient 3), −7 (constant)

Three terms: first is quadratic, second is linear, third is constant. Key Rule: Each term is separated by + or −.

What are the coefficients in: −x + 2y − 5z?Medium
Show Answer

Coefficient of x = −1; of y = 2; of z = −5

−x = −1 × x; 2y = 2 × y; −5z = −5 × z. Key Rule: The sign belongs to the coefficient.

How many variable terms are in: 3 + 2 + 7?Easy
Show Answer

Zero (no variable terms)

3, 2, and 7 are all constants. There are no letters in this expression. Key Rule: A purely numerical expression has no variable terms.

Describe the difference between a coefficient and a constant using the expression 5n + 8.Hard
Show Answer

Coefficient (5): multiplies the variable n. Constant (8): stands alone, no variable.

Coefficient tells us how many of the variable we have. Constant is fixed and does not depend on n. Key Rule: Coefficient is attached to variable; constant is independent.

SET D Challenge – Complex Expressions

In 2x + 3y + x + y − 4, find the coefficient of x after combining like terms.Hard
Show Answer

3

x terms: 2x + x = 3x. y terms: 3y + y = 4y. Constant: −4. Simplified: 3x + 4y − 4. Coefficient of x = 3. Key Rule: Combine like terms first, then identify coefficients.

How many terms does (3a + 2b)(a + b) have BEFORE expanding?Hard
Show Answer

It is a product of two binomials — each binomial has 2 terms.

Before expanding, (3a + 2b) and (a + b) are each binomials. The product itself is not yet a sum of terms. Key Rule: Count terms after expanding and simplifying.

In 5 + 3a, is 5 the coefficient or constant? Explain.Medium
Show Answer

5 is the constant.

5 has no variable attached. The coefficient is 3 (for the variable a). Key Rule: Coefficient attaches to a variable; constant stands alone.

Write an expression where the coefficient of x is greater than the constant.Medium
Show Answer

e.g., 10x + 3

Coefficient of x = 10; constant = 3. 10 > 3. Key Rule: Any expression with a larger coefficient than constant qualifies.

In an expression, the coefficient of x is 0. What does this mean?Hard
Show Answer

The x term vanishes (0 × x = 0), so x is not in the expression.

0 × x = 0 for any value of x. The term disappears. Key Rule: A zero coefficient eliminates the variable term.

In 4n + n + n, what is the total coefficient of n?Medium
Show Answer

6

4n + n + n = (4+1+1)n = 6n. Coefficient = 6. Key Rule: Combine like terms by adding their coefficients.

True or False: In 3x + 3, both 3s are coefficients.Hard
Show Answer

False

In 3x, the 3 is a coefficient (multiplies x). In +3, the 3 is a constant (no variable). Key Rule: Same number can play different roles depending on whether it multiplies a variable.

Write a polynomial with 4 terms: two variable terms with different variables, one quadratic term, and one constant.Hard
Show Answer
