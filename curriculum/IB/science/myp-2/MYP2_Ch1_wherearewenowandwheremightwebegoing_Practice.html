SET A Foundation - Basic Concepts

Define observation and inference with one example each.
Easy
Show Answer

Answer: Observation is directly sensed data; inference is interpretation based on evidence.

Write one testable scientific question about plant growth.
Easy
Show Answer

Answer: Example: How does sunlight duration affect bean plant height in 10 days?

Identify independent and dependent variables in: effect of fertilizer amount on plant height.
Easy
Show Answer

Answer: Independent: fertilizer amount; Dependent: plant height.

State two control variables for the plant experiment above.
Easy
Show Answer

Answer: Examples: same seed type, same water amount.

List the correct order of the scientific method steps.
Easy
Show Answer

Answer: Question -> Hypothesis -> Experiment -> Data -> Analysis -> Conclusion.

What is a fair test? Give one condition needed for fairness.
Medium
Show Answer

Answer: Only one variable is changed while others are kept constant.

Differentiate between accuracy and precision.
Medium
Show Answer

Answer: Accuracy is closeness to true value; precision is repeatability.

Write a hypothesis in if-then-because form for dissolving sugar in water.
Medium
Show Answer

Answer: If water temperature increases, then sugar dissolves faster because particle motion increases.

Give one qualitative and one quantitative observation for a burning candle.
Medium
Show Answer

Answer: Qualitative: flame is yellow. Quantitative: candle height decreased by 2 cm.

Why are repeated trials important in an experiment?
Hard
Show Answer

Answer: They reduce random error and improve reliability.

SET B Core Practice - Skill Building

A student measures mass as 21 g, 20 g, 19 g. Find mean mass.
Easy
Show Answer

Answer: Mean = (21+20+19)/3 = 20 g.

Classify each as physical or chemical change: melting ice, rusting iron, boiling water, burning paper.
Easy
Show Answer

Answer: Physical: melting, boiling. Chemical: rusting, burning.

A thermometer reads 24°C, 26°C, 25°C. Find range.
Easy
Show Answer

Answer: Range = 26 - 24 = 2°C.

Design a simple table to record seed germination over 7 days.
Medium
Show Answer

Answer: Include columns: Day, Seeds Germinated, Notes.

Suggest one safety precaution for heating liquids in a lab.
Medium
Show Answer

Answer: Wear goggles and point test tube away from people.

State one source of systematic error and one random error.
Medium
Show Answer

Answer: Systematic: miscalibrated balance. Random: reaction-time variation.

How would you improve reliability in a reaction-time experiment?
Hard
Show Answer

Answer: Use larger sample size and repeat multiple trials per person.

Convert 0.25 L to mL and 750 mL to L.
Medium
Show Answer

Answer: 0.25 L = 250 mL; 750 mL = 0.75 L.

Why should units always be included with measurements?
Medium
Show Answer

Answer: Units define magnitude and prevent misinterpretation.

Write a conclusion statement from data showing higher light gives taller plants.
Hard
Show Answer

Answer: Conclusion: increasing light intensity increased average plant height in this setup.

SET C Application - Context and Reasoning

Plan an experiment to test which material insulates best (cotton, foil, paper).
Medium
Show Answer

Answer: Use equal containers and compare temperature drop over fixed time.

Explain how to ensure fairness in the insulation experiment.
Medium
Show Answer

Answer: Keep water volume, starting temperature, and timing constant.

A class measured pulse before exercise: 72, 76, 80, 74. Find average pulse.
Easy
Show Answer

Answer: Average = 75 bpm.

After exercise pulses are 110, 120, 104, 116. Find average increase.
Medium
Show Answer

Answer: New average 112.5; increase = 37.5 bpm.

Create a bar graph plan to compare before/after pulse values.
Medium
Show Answer

Answer: Two bars per student or group means for before and after.

Interpret: if one group has very different data, what should you check?
Hard
Show Answer

Answer: Check measurement method, timing, and possible outlier causes.

Suggest one ethical consideration in human experiments.
Hard
Show Answer

Answer: Obtain informed consent and avoid harmful procedures.

Write one limitation and one improvement for pulse experiment.
Hard
Show Answer

Answer: Limitation: small sample. Improvement: more participants and repeated sessions.

Differentiate reliability and validity in one sentence each.
Hard
Show Answer

Answer: Reliability: consistent results. Validity: measures what it intends to measure.

Draft a short claim-evidence-reasoning conclusion from pulse data.
Hard
Show Answer

Answer: Claim: exercise raises pulse. Evidence: mean rose by 37.5 bpm. Reasoning: higher oxygen demand increases heart rate.

List variables and controls for your water-quality investigation.
Hard
