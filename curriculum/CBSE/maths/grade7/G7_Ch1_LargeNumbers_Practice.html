SET A Foundation — Place Value Identification

What is the place value of 7 in the number 47,32,915?
Easy
Show Answer

Answer: 7,00,000 (Seven lakhs)

Method: Place Value Chart
Write 47,32,915 in the Indian place value chart.
Crores | Ten-Lakhs | Lakhs | Ten-Thousands | Thousands | Hundreds | Tens | Ones
— | 4 | 7 | 3 | 2 | 9 | 1 | 5
The digit 7 is in the Lakhs place.
Key Rule: Place value = digit × value of its position. 7 × 1,00,000 = 7,00,000
What is the face value of 5 in 3,58,24,601?
Easy
Show Answer

Answer: 5

Method: Face Value Rule
Face value is always the digit itself, regardless of its position.
The digit 5 appears in the number — its face value is simply 5.
Key Rule: Face value = the digit as written. Place value ≠ face value (except in the ones place when digit ≠ 0).
