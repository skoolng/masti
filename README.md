# Algbrox Runner MVP

Client-side web MVP for algebra operation precedence practice with an endless-runner game loop.

## Features
- Temple-run style lane game with math gates and jump/slide obstacles
- Adaptive question generator across skill tracks:
  - `+/-`, `*//`, exponents/roots, grouping, mixed precedence, signed numbers
- Local persistence in `IndexedDB`:
  - profiles, settings, runs, attempts, mastery, quests, quest progress
- Roblox-inspired theme packs and avatar customization
- Math Antics resource links shown contextually and in a resources screen
- PWA-ready (`manifest.webmanifest`, `sw.js`) with offline caching

## Run
Open `index.html` in a browser, or serve the folder locally for full PWA behavior.
