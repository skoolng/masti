SET A Identify the AP

Which of the following is an AP? (A) 1, 2, 4, 8 (B) 3, 6, 9, 12 (C) 1, 1, 2, 3, 5 (D) 2, 5, 10, 17Easy
Show Answer

(B) 3, 6, 9, 12 — each term increases by 3, so d = 3.

Is the sequence 5, 8, 11, 14, 17 an AP? If yes, state a and d.Easy
Show Answer

Yes. a = 5, d = 3 (each consecutive difference = 3).

Check whether 0, −4, −8, −12 is an AP.Easy
Show Answer

Yes. d = −4 − 0 = −4. All consecutive differences are equal.

The sequence 1, 3, 9, 27, 81 — is this an AP? Explain.Easy
Show Answer

No. This is a geometric progression; differences are 2, 6, 18, 54 — not constant.

Which of these is NOT an AP? (A) 2, 2, 2, 2 (B) 0.5, 1, 1.5, 2 (C) 1, 2, 3, 5, 8 (D) −5, 0, 5, 10Easy
Show Answer

(C) 1, 2, 3, 5, 8 is the Fibonacci sequence; differences 1, 1, 2, 3 are not constant.

Is a constant sequence like 7, 7, 7, 7, 7 an AP?Easy
Show Answer

Yes. d = 0. Any constant sequence is an AP with common difference 0.

The sequence 100, 90, 80, 70, 60 — is it an AP? State d.Easy
Show Answer

Yes. d = 90 − 100 = −10.

Write the next two terms if the AP is: 4, 7, 10, 13, ____, ____Easy
Show Answer

d = 3. Next terms: 16, 19.

A student writes: 2, 4, 6, 8, 10, 13. Does this remain an AP throughout?Easy
Show Answer

No. The first five differences are 2, but the last gap is 3 (13 − 10 = 3), breaking the AP.

Determine whether the sequence of even numbers 2, 4, 6, 8, ... is an AP.Easy
Show Answer

Yes. d = 2. All even natural numbers form an AP.

SET B Verify and Justify

For the AP 3, 3+√2, 3+2√2, 3+3√2, find d.Medium
Show Answer

d = (3+√2) − 3 = √2.

Is 1/2, 1/3, 1/4, 1/5 an AP? Justify.Medium
Show Answer

No. Differences: 1/3 − 1/2 = −1/6; 1/4 − 1/3 = −1/12. Not equal.

Check if √2, 2√2, 3√2, 4√2 is an AP.Medium
Show Answer

Yes. d = √2. Each term increases by √2.

The sequence a, a+d, a+2d, a+3d — prove it is always an AP.Medium
Show Answer

Difference between any consecutive terms = (a+kd) − (a+(k−1)d) = d, which is constant. ✓

Is p, 2p, 3p, 4p (p ≠ 0) an AP? State d in terms of p.Medium
Show Answer

Yes. d = p.

The squares of integers: 1, 4, 9, 16, 25 — is this an AP?Medium
Show Answer

No. Differences: 3, 5, 7, 9 — these are not constant.

For what value of x is x+2, 2x−1, 3x−4 an AP?Medium
Show Answer

For AP: 2(2x−1) = (x+2)+(3x−4) → 4x−2 = 4x−2. True for all x — it is always an AP!

Check if cos²θ, 1, sin²θ forms an AP when θ = 45°.Medium
Show Answer

At θ=45°: cos²45° = 1/2, 1, sin²45° = 1/2. Check: 1−1/2 = 1/2 and 1/2−1 = −1/2. Not equal. Not an AP.

Does the sequence n², (n+1)², (n+2)² form an AP for any positive integer n?Hard
Show Answer

Differences: (n+1)²−n² = 2n+1 and (n+2)²−(n+1)² = 2n+3. These are not equal (differ by 2). Not an AP.

If a, b, c are in AP, show that 2b = a + c.Medium
Show Answer

Since b−a = c−b (constant difference), adding: 2b = a+c. ✓
