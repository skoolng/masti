### Chapter Summary
- classifying real numbers, including rational and irrational values
- rewriting numbers using powers, roots, surds and standard form
- using index laws and negative powers to express the same quantity in different ways
- working with accuracy, approximation and significant figures
- explaining why different forms of the same number can be useful in different contexts

### Set A - Foundation
1. Write 78,007 in expanded form.
2. Arrange these numbers in ascending order: 529, 9968, 55, 543.
3. Round 81472 to the nearest 100.
4. Find the value of 5^3.
5. Write 2/3 as a decimal.

### Set B - Core Practice
1. Use divisibility rules to decide whether 1947 is divisible by 3 and 9.
2. Find the least common multiple of 8 and 12.
3. Write 0.42 as a fraction in simplest form.
4. Classify sqrt(2) as rational or irrational.
5. Evaluate 2^0 and explain why the answer makes sense.

### Set C - Application
1. Write a rule for finding the next term in 4, 7, 10, 13, ... and give the 10th term.
2. Evaluate (6^2 - 5^2) / 1.
3. A box holds 38 packets. 19 boxes arrive. Find the total number of packets.
4. Explain why 0.4, 40% and 2/5 represent the same amount.
5. Choose a real-life quantity that is easier to write in standard form and explain why.

### Set D - Challenge
1. Use the chapter title 'In how many different ways can we express the same thing?' to write one sentence about how number forms can change the way information is communicated.
2. Show two different ways to represent the number 1200.
3. Evaluate sqrt(144) + 9.
4. Identify which is larger without a calculator: 5^4 or 4^5.
5. Find the missing number: 18, __, 30, 36 if the pattern adds the same amount each time.
