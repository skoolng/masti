SET A Identifying and Writing Linear Equations

Which of the following is a linear equation in two variables? (a) x² + y = 3 (b) 2x + 3y = 7 (c) xy = 4 (d) x/y = 2Easy
Show Answer

(b) 2x + 3y = 7. A linear equation in two variables has the form ax + by + c = 0 where both variables appear with degree 1.

Write the equation 3x = 5 – 2y in standard form ax + by + c = 0.Easy
Show Answer

3x + 2y – 5 = 0. Rearrange: 3x + 2y – 5 = 0, so a = 3, b = 2, c = –5.

Find two solutions of the equation 2x + y = 6.Easy
Show Answer

When x = 0: y = 6 → (0, 6). When x = 3: y = 0 → (3, 0). (Many solutions exist.)

If (2, k) is a solution of 4x – 3y = 5, find k.Easy
Show Answer

4(2) – 3k = 5 → 8 – 3k = 5 → 3k = 3 → k = 1.

Express y in terms of x for the equation 3x – 4y + 12 = 0.Easy
Show Answer

–4y = –3x – 12 → y = (3x + 12)/4 = (3/4)x + 3.

Is (1, –1) a solution of the equation 5x – 2y = 7?Medium
Show Answer

5(1) – 2(–1) = 5 + 2 = 7. Yes, (1, –1) satisfies the equation.

Find the value of k if x = 2, y = 1 is a solution of 2x + ky = 8.Medium
Show Answer

2(2) + k(1) = 8 → 4 + k = 8 → k = 4.

Write a linear equation in two variables that has (0, 5) and (5, 0) as solutions.Medium
Show Answer

x + y = 5 (or equivalently x + y – 5 = 0). Check: 0+5=5 ✓ and 5+0=5 ✓.

The cost of 3 notebooks and 2 pens is ₹90. Write this as a linear equation in two variables and find one solution.Medium
Show Answer

Let notebook = x, pen = y. Equation: 3x + 2y = 90. One solution: x = 10, y = 30 (since 30 + 60 = 90).

How many solutions does a linear equation in two variables have? Explain with an example.Hard
Show Answer

Infinitely many solutions. For example, x + y = 4 is satisfied by (0,4), (1,3), (2,2), (–1,5), etc. Every point on the line is a solution.

SET B Solutions and Verification

Find three solutions of 2x – y = 4.Easy
Show Answer

(0, –4), (2, 0), (3, 2). Check each: 2(0)–(–4)=4 ✓; 2(2)–0=4 ✓; 2(3)–2=4 ✓.

Express x in terms of y for 5x + 3y = 15.Easy
Show Answer

5x = 15 – 3y → x = (15 – 3y)/5 = 3 – 3y/5.

Check if (3, 2) is a solution of x + 2y = 7.Easy
Show Answer

3 + 2(2) = 3 + 4 = 7. Yes, (3, 2) is a solution.

For the equation 4x + 6y = 24, find the x-intercept and y-intercept.Easy
Show Answer

x-intercept (y=0): 4x = 24 → x = 6. y-intercept (x=0): 6y = 24 → y = 4.

If x = t and y = 2t – 1 for all values of t, find the linear equation satisfied by x and y.Medium
Show Answer

Substitute x = t: y = 2x – 1, i.e., y – 2x + 1 = 0 or 2x – y – 1 = 0.

Find k if 3x + ky = 12 passes through (4, 0) and find another point on the line.Medium
Show Answer

3(4) + k(0) = 12 → 12 = 12 ✓ for any k. Equation passes through (4,0) regardless. If k=3: another point is (0,4).

A linear equation passes through (–2, 5) and (4, 2). Write the equation.Medium
Show Answer

Slope = (2–5)/(4–(–2)) = –3/6 = –1/2. Using (4,2): y–2 = –½(x–4) → 2y–4=–x+4 → x+2y=8.

The sum of twice a number and three times another is 24. If the first number is 6, find the second.Medium
Show Answer

2(6) + 3y = 24 → 12 + 3y = 24 → 3y = 12 → y = 4.

If ax + by = a – b and bx – ay = a + b have solution x = 1, y = –1, verify this.Hard
Show Answer

Eq1: a(1)+b(–1)=a–b ✓. Eq2: b(1)–a(–1)=b+a=a+b ✓. Verified.

For what value of k does kx + 4y = k – 4 have infinitely many solutions when paired with 16x + ky = k?Hard
Show Answer

For infinitely many solutions: k/16 = 4/k = (k–4)/k. From k/16=4/k: k²=64, k=±8. Check k=8: 8/16=4/8=½ and (8–4)/8=½ ✓. k=8.

If 2x + 3y = k and (–1, 4) lies on this line, find k.Easy
Show Answer

2(–1) + 3(4) = –2 + 12 = 10. So k = 10.

Convert x/3 + y/4 = 1 into standard form.Medium
Show Answer

Multiply by 12: 4x + 3y = 12 → 4x + 3y – 12 = 0.

The perimeter of a rectangle is 40 cm. If length is l and breadth is b, write a linear equation and find l when b = 8.Medium
Show Answer

2(l + b) = 40 → l + b = 20. When b = 8: l = 12 cm.

Write an equation of the form ax + by = c such that a, b, c are consecutive integers and the line passes through the origin.Medium
Show Answer

For passing through origin, c = 0. Consecutive integers with c=0: not possible unless 0 is included. Example: a=1, b=2, c=0 → x + 2y = 0. Note: 0,1,2 are consecutive.

Show that x = 2, y = 3 satisfies ax + by = 2a + 3b for any values of a and b.Medium
Show Answer

LHS: a(2) + b(3) = 2a + 3b = RHS. This holds for all a and b, so (2,3) always satisfies the equation.

If (a, b) is a solution of x + y = 5 and (a, b) is also a solution of 2x – y = 1, find a and b.Hard

a + b = 5 and 2a – b = 1. Add: 3a = 6 → a = 2. Then b = 3.

Find values of p and q if px + 2y = 8 and 4x + qy = 16 represent the same line.Hard
Show Answer

For coincident lines: p/4 = 2/q = 8/16 = 1/2. So p = 2 and q = 4.

SET D Mixed Review and Challenge

How is a linear equation in one variable different from one in two variables?Easy
Show Answer

A linear equation in one variable (e.g., 2x=6) has a unique solution. In two variables (e.g., 2x+y=6), it has infinitely many solutions forming a line.

Write two different linear equations that have (3, –2) as a solution.Easy
Show Answer

x + y = 1 (since 3+(–2)=1 ✓) and x – y = 5 (since 3–(–2)=5 ✓).

Find the value of m if (m, 2m) lies on the line 3x – 4y + 5 = 0.Easy
Show Answer

3m – 4(2m) + 5 = 0 → 3m – 8m + 5 = 0 → –5m = –5 → m = 1.

A linear equation in two variables represents a _____ geometrically.Easy
Show Answer

A straight line. Every linear equation ax + by + c = 0 (with a and b not both zero) represents a straight line in the coordinate plane.

If x = a and y = b is a solution of both x – y = 2 and x + y = 4, what is the ordered pair (a, b)?Medium
Show Answer

Adding: 2x=6→x=3=a. Subtracting: 2y=2→y=1=b. Answer: (3,1).

For the equation 2x + 3y = 12, find points where the line crosses the axes and plot them.Medium
Show Answer

x-intercept: y=0 → 2x=12 → (6,0). y-intercept: x=0 → 3y=12 → (0,4). Plot and join these two points.

Determine if the points (0,3), (1,1), and (2,–1) are collinear using a linear equation.Medium
Show Answer

Slope between (0,3) and (1,1): –2. Equation: y=–2x+3 → 2x+y=3. Check (2,–1): 4+(–1)=3 ✓. All three are collinear.

The line 3x – 2y = k passes through the point of intersection of 4x + y = 11 and x + 3y = 9. Find k.Hard
Show Answer

Solve 4x+y=11 and x+3y=9: From first, y=11–4x. Sub: x+3(11–4x)=9→x+33–12x=9→–11x=–24→x=24/11. y=11–96/11=25/11. k=3(24/11)–2(25/11)=72/11–50/11=22/11=2. k=2.

Show that the equation x/a + y/b = 1 represents a line with x-intercept a and y-intercept b.Hard
Show Answer

At y=0: x/a=1 → x=a ✓. At x=0: y/b=1 → y=b ✓. This "intercept form" always has x-intercept a and y-intercept b.

If the line (a+b)x + (a–b)y = a²+b² passes through (1,1), find the relationship between a and b.Hard
Show Answer

(a+b)(1)+(a–b)(1)=a²+b² → a+b+a–b=a²+b² → 2a=a²+b² → a²–2a+b²=0 → (a–1)²+b²=1. This is a circle of radius 1 centered at (1,0).

Find the slope of the line 3x – 4y + 8 = 0.Easy
Show Answer

What is the slope of a horizontal line? Of a vertical line?Easy

Horizontal line: slope = 0. Vertical line: slope is undefined (no slope).

Find the slope and y-intercept of 2x + 5y = 10.Easy

Two lines have slopes 2 and –1/2. Are they perpendicular?Medium
Show Answer

Product of slopes = 2 × (–1/2) = –1. Yes, lines with slopes whose product is –1 are perpendicular.

Are the lines 2x + 3y = 6 and 4x + 6y = 15 parallel?Medium

Slope of first: –2/3. Slope of second: –4/6 = –2/3. Same slope, different intercepts → parallel.

Find the equation of the line with slope –3 passing through (2, 4).Medium

At what point does 4x – y = 8 cross the y-axis? Verify by substitution.Medium

If two lines have the same y-intercept but different slopes, where do they intersect?Hard
Show Answer

They intersect at the y-axis, i.e., at the point (0, c) where c is their common y-intercept.

A line cuts the x-axis at (–2, 0) and y-axis at (0, 3). Find the slope, write the equation, and determine if (4, 6) lies on it.Hard

Slope=(3–0)/(0–(–2))=3/2. Equation: y=3x/2+3 → 3x–2y+6=0. Check (4,6): 12–12+6=6≠0. (4,6) does not lie on it.

Three lines x + y = 6, 2x – y = 0, and x – 2y + 3 = 0 form a triangle. Find the vertices.Hard

Find the area of the triangle formed by 3x + 4y = 12, x = 0, and y = 0 using the intercept method.Hard

Intercepts: (4,0) and (0,3). Area = ½ × base × height = ½ × 4 × 3 = 6 sq. units.

For the system kx + 3y = k – 3 and 12x + ky = k, find k for which the system has infinitely many solutions.Hard

k/12 = 3/k = (k–3)/k. From k/12=3/k: k²=36→k=±6. Test k=6: 6/12=½, 3/6=½, 3/6=½ ✓. Test k=–6: –6/12=–½, 3/–6=–½, –9/–6=3/2≠–½. So k=6.

Two students each graph a system. Student A finds 2 solutions, Student B finds 1. Who is correct for a pair of linear equations, and why?Hard

First pair: 2x+2y=10 ≡ x+y=5 (same line → coincident → infinitely many). Second pair: 2x+2y=12 ≡ x+y=6 ≠ x+y=5 (parallel → no intersection → no solution).
