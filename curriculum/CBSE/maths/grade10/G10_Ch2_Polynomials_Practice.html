Set A Identifying Degree and Type

What is the degree of the polynomial p(x) = 7x³ − 3x² + 2x − 5?
Show Answer

Degree = 3 (highest power of x is 3). It is a cubic polynomial.

Easy
Classify 4x² − 9 by type and state its degree.
Show Answer

Degree = 2. It is a quadratic polynomial.

Easy
Is 2x + 5 a monomial, binomial, or trinomial? What is its degree?
Show Answer

It is a binomial (two terms). Degree = 1 (linear polynomial).

Easy
Write a polynomial of degree 0. What is such a polynomial called?
Show Answer

Example: p(x) = 8. A degree-0 polynomial is called a constant polynomial.

Easy
What is the degree of the zero polynomial?
Show Answer

The zero polynomial has no defined degree (sometimes written as −∞ or undefined).

Medium
Find the degree of p(x) = (x² + 3)(x³ − x + 1).
Show Answer

Degree = 2 + 3 = 5. (Leading term is x² · x³ = x⁵.)

Medium
How many terms does the polynomial 5x⁴ − 2x³ + x − 7 have? State its type by number of terms.
Show Answer

It has 4 terms, so it is a polynomial (specifically a degree-4 or biquadratic polynomial).

Easy
Is √2 · x + 3 a polynomial? If yes, state its degree.
Show Answer

Yes, it is a polynomial (coefficients need not be integers). Degree = 1.

Medium
Which of the following is NOT a polynomial: (a) x² + 1/x (b) x² + √3 (c) 2x³ − 4?
Show Answer

(a) x² + 1/x is NOT a polynomial because 1/x = x⁻¹ has a negative exponent.

Medium
A polynomial has degree 3, leading coefficient −2, and constant term 7. Write one such polynomial.
Show Answer

Example: p(x) = −2x³ + 5x² − x + 7. (Any cubic with leading coeff −2 and constant 7 is valid.)

Set B Coefficients and Leading Terms

What is the leading coefficient of p(x) = −3x⁴ + 7x² − 1?
Show Answer

Leading coefficient = −3.

Easy
Write the polynomial 3 + x − 2x² in standard form and state its degree.
Show Answer

Standard form: −2x² + x + 3. Degree = 2.

Easy
If p(x) = ax³ + bx + c and the coefficient of x² is zero, what does that tell you about the polynomial?
Show Answer

The polynomial has no x² term; the b in standard form ax³ + bx² + cx + d has b = 0 here. It is still cubic.

Medium
Find the coefficient of x in p(x) = (x + 2)(x − 3).
Show Answer

Expand: x² − 3x + 2x − 6 = x² − x − 6. Coefficient of x = −1.

Medium
What is the constant term of p(x) = (2x − 1)(x + 3)?
Show Answer

Expand: 2x² + 6x − x − 3. Constant term = −3.

Easy
For p(x) = x⁴ − 5x³ + 0·x² + 2x − 8, list all four coefficients in order.
Show Answer

a₄ = 1, a₃ = −5, a₂ = 0, a₁ = 2, a₀ = −8.

Medium
Two polynomials p(x) = 2x² + kx − 3 and q(x) = 2x² + 5x − 3 are equal for all x. Find k.
Show Answer

Equating coefficients of x: k = 5.

Medium
What is the degree of the product of a linear and a quadratic polynomial?
Show Answer

Degree = 1 + 2 = 3.

Easy
If p(x) has degree m and q(x) has degree n (m > n), what is the degree of p(x) + q(x)?
Show Answer

Degree of p(x) + q(x) = m (the higher degree dominates, assuming leading terms don't cancel).

Hard
Can the sum of two polynomials each of degree 3 have degree less than 3? Give an example.
Show Answer

Yes. E.g. p(x) = x³ + 2x and q(x) = −x³ + x + 1. Sum = 3x + 1, degree 1.

Set C Evaluating Polynomials

Find p(2) for p(x) = x³ − 4x + 1.
Show Answer

p(2) = 8 − 8 + 1 = 1.

Easy
Evaluate p(−1) for p(x) = 2x² − 3x + 5.
Show Answer

p(−1) = 2(1) − 3(−1) + 5 = 2 + 3 + 5 = 10.

Easy
If p(x) = x² − 5x + 6, find p(3) and p(2).
Show Answer

p(3) = 9 − 15 + 6 = 0; p(2) = 4 − 10 + 6 = 0. Both are zeroes!

Easy
For p(x) = 3x³ − 2x² + x − 4, compute p(0).
Show Answer

p(0) = 0 − 0 + 0 − 4 = −4. (p(0) always equals the constant term.)

Easy
Find p(1) + p(−1) for p(x) = x⁴ − x³ + x² − x + 1.
Show Answer

p(1) = 1 − 1 + 1 − 1 + 1 = 1; p(−1) = 1 + 1 + 1 + 1 + 1 = 5. Sum = 6.

Medium
If p(x) = x² + px + q and p(1) = 0, p(−2) = 0, find p and q.
Show Answer

p(1) = 1 + p + q = 0 → p + q = −1. p(−2) = 4 − 2p + q = 0 → −2p + q = −4. Subtract: 3p = 3, p = 1, q = −2. So p = 1, q = −2.

Hard
Evaluate p(√2) for p(x) = x² − 2.
Show Answer

p(√2) = (√2)² − 2 = 2 − 2 = 0.

Medium
For p(x) = 2x + 3, find the value of x for which p(x) = 0.
Show Answer

2x + 3 = 0 → x = −3/2.

Easy
If p(x) = x³ + ax² + bx + c and p(0) = −6, p(1) = 0, p(−1) = −8, find a, b, c.
Show Answer

c = −6; 1+a+b−6 = 0 → a+b = 5; −1+a−b−6 = −8 → a−b = −1. So a = 2, b = 3. Thus a=2, b=3, c=−6.

Hard
Find p(t²) for p(x) = x² − 3x + 2.
Show Answer

p(t²) = (t²)² − 3(t²) + 2 = t⁴ − 3t² + 2.

Medium

Set D Mixed / HOTS

Write a polynomial of degree 4 that has exactly three terms.
Show Answer

Example: p(x) = x⁴ − 5x² + 4 (three terms, degree 4 — a trinomial).

Medium
If 2 is a zero of p(x) = x² − kx + 6, find k.
Show Answer

p(2) = 4 − 2k + 6 = 0 → 2k = 10 → k = 5.

Medium
A polynomial p(x) of degree 2 satisfies p(0) = 1, p(1) = 3, p(−1) = 1. Find p(x).
Show Answer

Let p(x) = ax² + bx + c. c = 1; a+b+1 = 3 → a+b = 2; a−b+1 = 1 → a−b = 0. So a = 1, b = 1. p(x) = x² + x + 1.

Hard
How many zeroes can a quadratic polynomial have at most? Why?
Show Answer

A quadratic polynomial can have at most 2 zeroes, because a degree-n polynomial has at most n real zeroes.

Easy
True or False: Every linear polynomial has exactly one zero. Justify.
Show Answer

True. p(x) = ax + b (a ≠ 0). Setting ax + b = 0 gives x = −b/a, exactly one solution.

Easy
The polynomial p(x) = x² − (k+1)x + k has x = k as a zero. Find k.
Show Answer

p(k) = k² − (k+1)k + k = k² − k² − k + k = 0. True for all k. (k is always a zero.)

Hard
If p(x) = x⁴ − 16, express it as a product of two quadratics.
Show Answer

x⁴ − 16 = (x²)² − 4² = (x² + 4)(x² − 4) = (x²+4)(x+2)(x−2).

Medium
For p(x) = 3x² + 5x − 2, verify that x = 1/3 is a zero.
Show Answer

p(1/3) = 3(1/9) + 5(1/3) − 2 = 1/3 + 5/3 − 2 = 6/3 − 2 = 2 − 2 = 0 ✓.

Medium
Can a polynomial of odd degree have no real zeroes? Explain.
Show Answer

No. By the Intermediate Value Theorem, an odd-degree polynomial with real coefficients must cross the x-axis at least once, guaranteeing at least one real zero.

Hard
Find the value of p(x) = x⁵ − x⁴ + x³ − x² + x − 1 at x = 1.
Show Answer

p(1) = 1 − 1 + 1 − 1 + 1 − 1 = 0.

Set A Finding Zeroes by Factoring

Find the zeroes of p(x) = x² − 5x + 6.
Show Answer

x² − 5x + 6 = (x − 2)(x − 3) = 0. Zeroes: x = 2, x = 3.

Easy
Find the zeroes of p(x) = x² − 9.
Show Answer

(x − 3)(x + 3) = 0. Zeroes: x = 3, x = −3.

Easy
Find the zero of the linear polynomial p(x) = 4x − 8.
Show Answer

4x − 8 = 0 → x = 2.

Easy
Find the zeroes of p(x) = x(x − 5).
Show Answer

Zeroes: x = 0 and x = 5.

Easy
Find the zeroes of p(x) = 2x² + 7x + 3.
Show Answer

2x² + 7x + 3 = (2x + 1)(x + 3) = 0. Zeroes: x = −1/2, x = −3.

Medium
How many zeroes does p(x) = (x − 1)² have? Are they distinct?
Show Answer

p(x) has one repeated zero x = 1 (multiplicity 2). They are not distinct.

Medium
Find zeroes of p(x) = 3x² − x − 2.
Show Answer

3x² − x − 2 = (3x + 2)(x − 1) = 0. Zeroes: x = 1, x = −2/3.

Medium
Find the zeroes of p(x) = x² + 1. How many real zeroes does it have?
Show Answer

x² + 1 = 0 → x² = −1. No real zeroes (discriminant = −4 < 0).

Medium
Find the zeroes of p(x) = x³ − x = x(x² − 1).
Show Answer

x(x − 1)(x + 1) = 0. Zeroes: x = 0, 1, −1.

Medium
If one zero of p(x) = x² − 7x + k is 4, find k and the other zero.
Show Answer

p(4) = 16 − 28 + k = 0 → k = 12. Other zero = 7 − 4 = 3. (Verify: 4 × 3 = 12 = k ✓)

Set B Using the Factor Theorem

State the Factor Theorem. When is (x − a) a factor of p(x)?
Show Answer

(x − a) is a factor of p(x) if and only if p(a) = 0.

Easy
Is (x − 2) a factor of p(x) = x³ − 4x² + 5x − 2?
Show Answer

p(2) = 8 − 16 + 10 − 2 = 0. Yes, (x − 2) is a factor.

Easy
Check if (x + 1) is a factor of x³ + 2x² − x − 2.
Show Answer

p(−1) = −1 + 2 + 1 − 2 = 0. Yes, (x + 1) is a factor.

Easy
Is (x − 3) a factor of x² + x − 12?
Show Answer

p(3) = 9 + 3 − 12 = 0. Yes.

Easy
Find k if (x − 2) is a factor of kx² − 3x + 2.
Show Answer

p(2) = 4k − 6 + 2 = 0 → 4k = 4 → k = 1.

Medium
Find k if (x + 1) is a factor of x³ + kx² + 3x + 2.
Show Answer

p(−1) = −1 + k − 3 + 2 = 0 → k = 2.

Medium
If both (x − 1) and (x + 2) are factors of p(x) = x³ + ax² + bx − 2, find a and b.
Show Answer

p(1) = 1+a+b−2 = 0 → a+b = 1. p(−2) = −8+4a−2b−2 = 0 → 4a−2b = 10 → 2a−b = 5. Adding: 3a = 6, a = 2, b = −1. a = 2, b = −1.

Hard
Show that (x − 1) is a factor of xⁿ − 1 for any positive integer n.
Show Answer

Let p(x) = xⁿ − 1. p(1) = 1ⁿ − 1 = 0. By Factor Theorem, (x − 1) is a factor for all positive integers n.

Hard
For what value of a is (x − a) a factor of x² − (a+1)x + a?
Show Answer

p(a) = a² − (a+1)a + a = a² − a² − a + a = 0. True for all values of a.

Hard
Use the Factor Theorem to show that (2x − 1) is a factor of 2x³ − x² + 2x − 1.
Show Answer

p(1/2) = 2(1/8) − (1/4) + 2(1/2) − 1 = 1/4 − 1/4 + 1 − 1 = 0. So (2x − 1) is a factor.

Medium

Set C Remainder Theorem

State the Remainder Theorem.
Show Answer

When p(x) is divided by (x − a), the remainder is p(a).

Easy
Find the remainder when x³ − 3x² + 4x − 1 is divided by (x − 2).
Show Answer

p(2) = 8 − 12 + 8 − 1 = 3.

Easy
Find the remainder when 2x² − 5x + 3 is divided by (x + 1).
Show Answer

p(−1) = 2 + 5 + 3 = 10.

Easy
Find the remainder when x⁴ + x³ − 2x² + 3 is divided by (x − 1).
Show Answer

p(1) = 1 + 1 − 2 + 3 = 3.

Easy
If the remainder when p(x) = x² + ax + b is divided by (x − 1) is 2, and by (x + 1) is 4, find a and b.
Show Answer

p(1) = 1+a+b = 2 → a+b = 1. p(−1) = 1−a+b = 4 → −a+b = 3. Subtract: 2a = −2, a = −1, b = 2. a = −1, b = 2.

Hard
Find the remainder when x¹⁰⁰ − 1 is divided by (x − 1).
Show Answer

p(1) = 1¹⁰⁰ − 1 = 0. Remainder = 0.

Medium
For what value of k does x³ − kx² + 2x − 1 leave remainder 2 when divided by (x − 1)?
Show Answer

p(1) = 1 − k + 2 − 1 = 2 → 2 − k = 2 → k = 0.

Medium
Find the remainder when p(x) = x² − 5x + 6 is divided by (x − 2). What does the answer tell you?
Show Answer

p(2) = 4 − 10 + 6 = 0. Remainder = 0 means (x − 2) is a factor of p(x).

Medium
The remainder when 2x³ + mx² − 5 is divided by (x − 1) equals the remainder when it is divided by (x + 1). Find m.
Show Answer

p(1) = 2+m−5 = m−3; p(−1) = −2+m−5 = m−7. Setting equal: m−3 = m−7 gives −3 = −7, contradiction. Re-examine: with constant −5, p(x) = 2x³+mx²+0·x−5. p(1)=2+m−5=m−3; p(−1)=−2+m−5=m−7. Equal: m−3=m−7 → no solution unless there's an x term. If p(x)=2x³+mx²+nx−5 and the condition is possible, add x term. As stated, no valid m exists (remainder theorem shows the remainders differ by 4 for any m).

Hard
Find the remainder when 3x³ − 4x² + 7x − 5 is divided by (3x − 1).
Show Answer

Remainder = p(1/3) = 3(1/27) − 4(1/9) + 7(1/3) − 5 = 1/9 − 4/9 + 21/9 − 45/9 = (1−4+21−45)/9 = −27/9 = −3.

Set D Mixed Zeroes Problems

Can a non-zero constant polynomial have any zeroes?
Show Answer

No. A constant polynomial p(x) = c (c ≠ 0) never equals zero, so it has no zeroes.

Easy
The zeroes of p(x) = ax² + bx + c are both zero. What can you conclude about b and c?
Show Answer

If both zeroes are 0: Sum = 0 = −b/a → b = 0. Product = 0 = c/a → c = 0. So b = 0 and c = 0.

Medium
Find zeroes of p(x) = x² − (√3 + 1)x + √3.
Show Answer

Factor: (x − √3)(x − 1) = 0. Zeroes: x = √3 and x = 1.

Hard
If one zero of 3x² − 8x + k is the reciprocal of the other, find k.
Show Answer

Let zeroes be α and 1/α. Product = α · (1/α) = 1 = k/3. So k = 3.

Hard
Find zeroes of p(x) = 6x² − 7x − 3.
Show Answer

6x² − 7x − 3 = (3x + 1)(2x − 3). Zeroes: x = −1/3 and x = 3/2.

Medium
The product of zeroes of p(x) = −3x² + 4x + 7 is ______.
Show Answer

Product = c/a = 7/(−3) = −7/3.

Medium
Find the number of real zeroes of p(x) = x² + 4x + 5 using the discriminant.
Show Answer

Discriminant = 16 − 20 = −4 < 0. No real zeroes.

Medium
If α and β are zeroes of 2x² − 5x + 3, find 1/α + 1/β.
Show Answer

1/α + 1/β = (α+β)/(αβ) = (5/2)/(3/2) = 5/3.

Hard
Verify that 2 and −3 are zeroes of p(x) = x² + x − 6.
Show Answer

p(2) = 4+2−6 = 0 ✓; p(−3) = 9−3−6 = 0 ✓. Both verified.

Easy
Find zeroes of p(x) = 4x² − 4x − 3.
Show Answer

(2x − 3)(2x + 1) = 0. Zeroes: x = 3/2 and x = −1/2.

Set A Reading Zeroes from Graphs (Conceptual)

The graph of y = p(x) crosses the x-axis at x = −2 and x = 4. What are the zeroes of p(x)?
Show Answer

Zeroes: x = −2 and x = 4. Where the graph crosses/touches the x-axis gives the zeroes.

Easy
How many zeroes does a polynomial have if its graph does not intersect the x-axis?
Show Answer

Zero real zeroes. No x-intercepts means no real zeroes (could have complex zeroes).

Easy
A parabola touches the x-axis at exactly one point (x = 3). What are the zeroes of this quadratic?
Show Answer

There is one repeated zero: x = 3 (multiplicity 2). The quadratic is of the form a(x−3)².

Easy
The graph of a cubic polynomial p(x) cuts the x-axis at x = −1, 0, and 2. Write p(x) assuming leading coefficient 1.
Show Answer

p(x) = (x + 1)(x)(x − 2) = x(x+1)(x−2) = x³ − x² − 2x.

Medium
How many x-intercepts can the graph of a quadratic polynomial have?
Show Answer

A quadratic can have 0, 1, or 2 x-intercepts depending on the discriminant (negative, zero, positive).

Easy
For y = (x−1)²(x+2), describe the behaviour at each x-intercept: does the graph cross or touch the axis?
Show Answer

At x = 1 (multiplicity 2): the graph touches the x-axis (bounces). At x = −2 (multiplicity 1): the graph crosses the x-axis.

Hard
What is the geometric interpretation of a repeated zero?
Show Answer

A repeated zero means the graph is tangent to the x-axis at that point — it touches but does not cross.

Medium
A graph of y = p(x) has x-intercepts at −3, 1, and 5. What is the minimum degree of p(x)?
Show Answer

Minimum degree = 3 (one factor for each distinct zero: (x+3)(x−1)(x−5)).

Medium
The graph of a polynomial passes through (0, −6) and has zeroes at 1, 2, and 3. Find p(x) with leading coefficient 1.
Show Answer

p(x) = (x−1)(x−2)(x−3). p(0) = (−1)(−2)(−3) = −6 ✓. So p(x) = (x−1)(x−2)(x−3).

Hard
Can a degree-4 polynomial graph touch but not cross the x-axis at two different points?
Show Answer

Yes. Example: p(x) = (x−1)²(x+2)² touches at x=1 and x=−2 without crossing.

x-intercepts: x = 7 and x = −3.

The graph y = ax² + bx + c has x-intercepts at 1/2 and −4. Write p(x) with integer coefficients.

p(x) = (2x−1)(x+4) = 2x² + 7x − 4.

For y = x² − 2x − 3, identify x-intercepts and state whether the parabola opens up or down.

Medium
Sketch the rough shape of y = x(x−4) and label x-intercepts and vertex x-coordinate.

x-intercepts: 0 and 4. Vertex x-coordinate: (0+4)/2 = 2. Parabola opens upward.

A polynomial graph crosses the x-axis exactly once and the curve goes from −∞ to +∞. What type of polynomial could it be?

A linear polynomial (degree 1) or an odd-degree polynomial with only one real zero.

Hard
The graph of y = p(x) is entirely above the x-axis. What does this say about the zeroes?

p(x) > 0 for all real x means p(x) has no real zeroes.

Write a quadratic polynomial whose graph touches the x-axis at x = −5 and opens downward.

p(x) = −(x+5)² = −x² − 10x − 25.

Set C Graphs of Linear and Quadratic Polynomials

What shape is the graph of a linear polynomial y = mx + c?
Show Answer

A straight line with slope m and y-intercept c.

Easy
What shape is the graph of any quadratic polynomial?
Show Answer

A parabola.

Easy
For y = x² − 6x + 9, how many x-intercepts are there? Find them.
Show Answer

(x−3)² = 0. Exactly one x-intercept at x = 3 (tangent to x-axis).

Easy
For y = −x² + 4, find x-intercepts and describe the graph.
Show Answer

x-intercepts: ±2. Opens downward. Vertex at (0, 4).

Easy
Without drawing, determine how many times y = x² + 6x + 10 meets the x-axis.
Show Answer

Discriminant = 36 − 40 = −4 < 0. The graph never meets the x-axis (0 intercepts).

Medium
The parabola y = ax² + bx + c passes through (1,0), (−3, 0), and (0, −3). Find a, b, c.
Show Answer

Zeroes 1 and −3: y = a(x−1)(x+3). Using (0,−3): −3 = a(−1)(3) = −3a → a = 1. So y = x²+2x−3.

Hard
At what point does the graph of y = 3x − 9 cross the x-axis?
Show Answer

3x − 9 = 0 → x = 3. The graph crosses at (3, 0).

Easy
Two quadratics share the same x-intercepts x = 2 and x = −1. How are the two polynomials related?
Show Answer

They are scalar multiples of each other: p(x) = k(x−2)(x+1) for different values of k. Same zeroes, different shapes (widths/directions).

Hard
For y = 2(x−1)(x−5), find the axis of symmetry (x-coordinate of vertex).
Show Answer

Axis of symmetry x = (1+5)/2 = 3.

Medium
A quadratic polynomial has no real zeroes. Its graph is fully below the x-axis. What is the sign of its leading coefficient?
Show Answer

The leading coefficient is negative (parabola opens downward, entirely below x-axis).

Medium

Set D HOTS – Graphs and Zeroes

A cubic polynomial has three distinct real zeroes. Roughly describe the shape of its graph.
Show Answer

The graph crosses the x-axis at three distinct points, with two local turning points (one local max, one local min).

Medium
Can a degree-2 polynomial have three x-intercepts? Explain.
Show Answer

No. A polynomial of degree n can have at most n real zeroes. A degree-2 polynomial has at most 2 x-intercepts.

Easy
The graph of y = p(x) has x-intercepts at −2, 1, and 4, and passes through (0, 8). Find p(x).
Show Answer

p(x) = k(x+2)(x−1)(x−4). p(0) = k(2)(−1)(−4) = 8k = 8 → k = 1. p(x) = (x+2)(x−1)(x−4).

Hard
Describe how the graph of y = (x−a)² compares to y = (x−a).
Show Answer

y = (x−a) is a line crossing the x-axis at a. y = (x−a)² is a parabola that touches (doesn't cross) the x-axis at a, always ≥ 0.

Hard
If p(x) = x³ − 6x² + 11x − 6, and you know x=1 is a zero, find all x-intercepts by factoring.
Show Answer

Divide by (x−1): x²−5x+6 = (x−2)(x−3). Zeroes: x = 1, 2, 3.

Hard
What does it mean geometrically when p(a) > 0?
Show Answer

The graph of y = p(x) is above the x-axis at x = a.

Easy
