Set AFoundation — Identifying & Defining Rational Numbers

Define a rational number. Give three examples.
Easy
Show Answer

Answer: A rational number is any number that can be written as p/q where p and q are integers and q ≠ 0.

Examples: 3/4, −5/2, 7 (= 7/1)

Is −7 a rational number? Justify.
Easy
Show Answer

Answer: Yes.

−7 = −7/1, which is of the form p/q with q ≠ 0. Key Rule: Every integer is rational.

Is 0 a rational number? Explain.
Easy
Show Answer

Answer: Yes — 0 = 0/1.

Key Rule: 0 is rational because it can be expressed as p/q with any non-zero q.

Write two rational numbers between 1 and 2.
Easy
Show Answer

Answer: 3/2 and 5/4 (many answers possible).

Between 1 = 4/4 and 2 = 8/4 → 5/4, 6/4, 7/4 all work.

State whether 0.75 is rational and express it as p/q.
Easy
Show Answer

Answer: 0.75 = 75/100 = 3/4. Yes, rational.

List all integers from −3 to 3 and express each as a rational number.
Easy
Show Answer

Answer: −3/1, −2/1, −1/1, 0/1, 1/1, 2/1, 3/1.

Which of these are rational: 1/2, √4, √5, −3/7?
Easy
Show Answer

Answer: 1/2 ✓, √4 = 2 ✓, −3/7 ✓. √5 is irrational.

Express 2.5 as a fraction in lowest terms.
Easy
Show Answer

How many rational numbers exist between 0 and 1?
Easy
Show Answer

Easy
